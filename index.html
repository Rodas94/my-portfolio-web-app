PROJECT 01

ASTU Exam URL Centralization

Ubuntu Apache PHP SEB

🏛️ Situation

Adama Science and Technology University (ASTU) utilizes multiple computer labs, including advanced smart labs, to administer critical electronic examinations. To manage these facilities, the university employs a hierarchical network architecture segmented by VLANs. However, because of this complex setup, IT staff must manually configure the Safe Exam Browser (SEB) on every individual machine—a tedious process prone to human error and operational inefficiencies.

🎯 Task

Create a single centralized access point to dynamically manage exam URLs across all labs and VLANs.

🛠️ Action

  • Deployed secure Apache + PHP + MySQL web server on Ubuntu
  • Built dynamic URL mapping system with automatic subnet detection
  • Created master SEB configuration pointing to central server

PROJECT 02

ASTU Modern Captive Portal with Keycloak Google SSO

OPNsense Keycloak Ubuntu SSO

🏛️ Situation

Adama Science and Technology University (ASTU) operates a large hierarchical campus network with multiple VLANs serving thousands of students, faculty, and staff. Previously, the network lacked proper access control — users could connect to the LAN and access the internet without any authentication. This created security risks, poor accountability, and inefficient bandwidth usage. The university required a modern, user-friendly captive portal solution that enforces institutional authentication using Google Workspace accounts (@astu.edu.et).

🎯 Task

As the System Administrator, I was tasked with designing and implementing a secure, scalable Captive Portal solution that:

  • Enforce authentication before granting internet access
  • Support seamless Google Workspace SSO via Keycloak
  • Deliver a branded, professional captive portal experience
  • Support institutional login and guest/voucher access
  • Integrate cleanly with the existing OPNsense firewall infrastructure

🛠️ Action

  • Deployed Keycloak + Google OAuth2 on Ubuntu
  • Custom branded responsive captive portal (HTML/CSS/JS)
  • Integrated with OPNsense firewall rules and VLANs

🏆 Result

100% authentication enforcement • Zero-trust model implemented

PROJECT 03

ASTU Application Share

PHP Samba Ubuntu Apache

🏛️ Situation

ASTU relies on a complex, hierarchical campus network segmented across multiple Virtual Local Area Networks (VLANs) to maintain traffic isolation and institutional security. Academic and administrative departments required a centralized, seamless solution to access shared institutional files hosted on private storage nodes without relying on third-party cloud services or exposing storage infrastructure directly to end-user devices.

🎯 Task

  • Objective: Design, implement, and deploy a responsive web-based File Share Portal.
  • Secure interface: Connect with back-end Samba (smbclient) shared storage from an isolated web tier.
  • Usability: Provide an intuitive, real-time searchable grid-view for non-technical users.
  • Branding: Enforce ASTU identity using a localized palette (#003366, Red, White).

🛠️ Action

  • PHP middleware with smbclient abstraction layer
  • Modern responsive grid UI with live search
  • Robust path handling and security isolation

🏆 Result

Fast, secure, zero-dependency internal file portal • Full institutional branding alignment
